# Heuristics

## Backend signals
Strong Laravel signals:
- `artisan`
- `bootstrap/app.php`
- `config/app.php`
- `routes/web.php` or `routes/api.php`
- `database/migrations/`

## Frontend signals
Strong Next.js signals:
- `next.config.*`
- `app/` with route segments and layout files
- `pages/` with Next-specific patterns

Strong Vite/React signals:
- `vite.config.*`
- `src/main.*`
- React dependencies in `package.json`
