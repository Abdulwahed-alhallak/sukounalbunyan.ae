# Stack Signatures

## Laravel + integrated frontend
Evidence usually includes Laravel directories plus `package.json` and `vite.config.*`.

## Laravel API + React/Vite
Evidence usually includes Laravel backend structure plus separate frontend source with Vite.

## Laravel API + Next.js
Evidence usually includes Laravel backend structure plus `next.config.*` and app or pages routes.

## Standalone Next.js
Evidence usually includes `next.config.*` and package scripts for `next dev`, `next build`, `next start`.

## Standalone React/Vite
Evidence usually includes `vite.config.*`, React dependencies, and an entry file like `src/main.tsx`.
