# 👥 موديول الموارد البشرية (HRM Module Deep Dive) v4.0.1

يعتبر موديول الموارد البشرية المحرك البشري لمنصة Noble. تم تحديثه في إبريل 2026 ليتوافق مع معايير **Vercel Geist 2026** مع دعم كامل للغة العربية.

## 1. التحديثات الاستراتيجية 2026
- **توطين شامل (Localization):** تمت ترجمة كافة الكيانات (الحضور، الرواتب، الإجازات) للغة العربية الفصحى لضمان تجربة مستخدم احترافية.
- **دعم RTL:** التحقق من كافة الهوامش والحواف (`ms-`, `me-`) لضمان محاذاة بصرية مثالية عند التحويل للغة العربية.
- **هوية بصرية موحدة:** استخدام مكونات `Sidebar` المحدثة مع Tooltips عربية ذكية وتحسين تباين الألوان.

## 2. المعمارية ودورة حياة الموظف (Employee Lifecycle)
يقوم موديول الموارد البشرية بالتعامل مع عدة كيانات تشكل معاً مسار الموظف بالمؤسسة:
1. **الأقسام (Departments):** تم توحيد المصطلحات ليتم استخدام Department بدلاً من Specialty.
2. **الحضور والانصراف (Attendance):** تم إصلاح العناوين والترجمات لضمان عرض دقيق للمديرين والموظفين.
3. **الرواتب (Payroll):** نظام احتساب ذكي يعتمد على الساعات الفعلية والبدلات المسجلة.

## 3. التكامل مع واجهة المستخدم (Frontend - React/Inertia)
تعتمد الواجهة على `Inertia.js` لتمرير البيانات من `EmployeeController` إلى مكونات React (مثل `Hrm/Employee/Index.tsx`). يتم استخدام نظام الجداول الموحد مع دعم كامل للفرز والبحث باللغة العربية.
